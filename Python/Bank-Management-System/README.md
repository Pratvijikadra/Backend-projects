# Bank Management System (OOP Based)

This project demonstrates all major OOP concepts in Python:

- Abstraction
- Encapsulation
- Inheritance
- Polymorphism
- Static Method
- Class Method

Explanation : This is a Bank Management System built using all major OOP concepts.
The Account class is abstract and defines common banking functionality.
SavingsAccount and CurrentAccount inherit from it and override the account_type method.
Balance is kept private using encapsulation.
Class method modifies bank name.
Static method defines bank rules.
